# estudo-comparativo-oncogenes-tsgs
Esse projeto visa fazer uma abordagem computacional para o estudo topológico comparativo de Oncogenes e Genes Supressores Tumorais em Redes de Interação Gênica. A rede de interação gênica usada foi a Reactome e a classifição de genes em Oncogenes e Genes Supressores Tumorais utilizada foi da COSMIC. O projeto foi realizado na linguagem de programação Python.
